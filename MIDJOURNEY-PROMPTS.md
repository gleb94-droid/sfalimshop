# Midjourney Prompts — Sfalim Shop ☕

> 12 ready-to-paste prompts, all locked to the brand orange `#FF6B35` and the dark `#0f0f0f` charcoal. No text inside — we'll add Hebrew/English manually in editing.

## How to use these

1. **Copy** the prompt under each title.
2. **Paste** into Midjourney (Discord or web).
3. **Run 2–3 times** per prompt — MJ varies. Best result usually comes after a few tries.
4. **Like one?** Add `--seed <number>` to the next run to get variations in the same style.
5. **Format tip:** prompts with `--ar 4:5` are vertical (good for Instagram posts and Reels covers); `--ar 1:1` are square (feed posts); `--tile` returns seamless patterns.

# 🔧 Power-User Tips

**Parameter cheat sheet:**
| Parameter | What it does | When to change |
|-----------|-------------|----------------|
| `--ar 1:1` | Square (Instagram feed) | Default for posts |
| `--ar 4:5` | Vertical (Instagram preferred) | Lifestyle / hero shots |
| `--ar 9:16` | Tall (story / Reel cover) | Story / Reels |
| `--style raw` | Less "MJ-styled", more photo-realistic feel | Editorial illustrations |
| `--stylize 50-1000` | Lower = literal, higher = artistic | Default 250-400 here |
| `--tile` | Seamless repeating pattern | Backgrounds only |
| `--seed <number>` | Same starting point | After you find a winner |
| `--no <thing>` | Negative prompt | e.g. `--no text, hands` |

**If MJ tries to add text:** Add `--no text, words, letters, typography` at the end.

**If the orange isn't strong enough:** Replace `warm orange #FF6B35` with `vivid bright orange (#FF6B35), high saturation, dominant color`.

**If you want it more illustrated and less "AI":** Add `, in the style of a printed children's book illustration` or `, in the style of a vintage editorial magazine` — gives MJ a clearer style reference.

**Combining the brand:** Once you find a great image, the Sfalim Shop logo (`public/exports/logo.png`) can be added in any editor (Canva, Photoshop, even Instagram's overlay tools) at the bottom or as a small mark.
